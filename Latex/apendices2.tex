\chapter{Plataformas Computacionais}

\section{Gráficos}


Fazer gráfico é uma arte. Um gráfico tem que ser bonito, elegante, conciso e mostrar o conteúdo de maneira simples e direta. Um gráfico leva uma mensagem. Se o leitor não capturar essa mensagem, o gráfico é inútil. Atualmente há inúmeros pacotes computacionais específicos para a confecção de gráficos, tanto pagos como gratuitos. Em geral, as opções pagas possuem uma interface gráfica, o que facilita bastante o processo de fazer um gráfico. Já as opções gratuitas em geral não possuem interface gráfica, o que afasta usuários não inclinados a programação. 


Algumas plataformas pagas mais tradicionais são:
\begin{itemize}

\item \textbf{Origin} - \url{www.microcal.com} (Windows): é o software mais utilizado no mundo pela comunidade acadêmica (de física pelo menos) para confecção de gráficos. É o Word dos gráficos. É um software pago, com interface gráfica e com inúmeros recursos. 

\item \textbf{Igor} - \url{www.wavemetrics.com} (Windows e Macintosh): é um software também comercial e interface gráfica porém segue uma filosofia diferente. É um concorrente direto do Origin, porém perde de longe em popularidade.

\item \textbf{Excel}: mundialmente famoso por planilhas mas também tem recursos poderosos para fazer gráficos e cálculos diversos, inclusive os mais complexos.

\item \textbf{Matlab} - \url{www.mathworks.com} (Windows, Macintosh e Linux): trata-se de uma linguagem de programação desenvolvida para modelagem matemática. Por ser uma plataforma comercial, há inúmeros recursos que a tornam extremamente versátil, simples, intuitiva e poderosa para a solução de problemas matemáticos e confecção de gráficos 2D, curvas de nível, superfície e 3D.

\end{itemize}

Existem outros títulos comerciais (Qtiplot para Windows, Kaleidagraph, MathCad, Maple, Mathematica, etc...) que seguem filosofias diversas. Já dentre as plataformas gratuitas destacamos:
\begin{itemize}
\item \textbf{Python}: é uma linguagem de programação poderosa, versátil e muito popular, desenvolvida para ser simples. Serve tanto para modelagem matemática quanto para programação em geral. Python conta com inúmeros recursos para confecção de gráficos, tornando-a uma ótima opção gratuita para fazer gráficos. A desvantagem (para alguns) é que não tem interface gráfica.

\item \textbf{Gnuplot}: é uma linguagem de programação específica para confecção de gráficos. Não é popular mas é tem uma comunidade fiel.

\item \textbf{SciDavis}: é uma cópia do Origin porém gratuita. Por ser gratuita não há um suporte e assim tem muitos problemas.

\end{itemize}

Essas versões gratuitas podem serem usadas em Windows, Macintosh e Linux. Há várias outras opções gratuitas (Octave, Qtiplot para Linux, etc...) porém menos utilizadas. Para encontrar documentação sobre qualquer uma dessas opções, basta procurar no Google.
